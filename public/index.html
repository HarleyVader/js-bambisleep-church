🔌 Connecting Your Toy

What is Intiface Central?

Intiface Central is a free, open-source desktop app by Nonpolynomial Labs that acts as a local server between your browser and Bluetooth/USB toys. It must be running on the same computer as your browser before you click “Connect” here.

Quick-Start (3 steps)

  1. Download & install Intiface Central from intiface.com/central (Windows / macOS / Linux).
  2. Start the server — open Intiface Central and press the big green Start Server button. The default address is ws://localhost:12345.
  3. Connect here — leave the address as ws://localhost:12345 and click Connect. Then click Scan to find your paired Bluetooth toy.

Pairing your toy first

Toys must be paired with your OS Bluetooth settings before Intiface Central can see them. Open your system’s Bluetooth & devices panel, put the toy in pairing mode (usually hold the power button 3–5 s until it flashes), and pair it. You only need to do this once.

Supported toy brands

Lovense, We-Vibe, Kiiroo, Lelo, Vibease, OhMiBod, Satisfyer, The Handy, and hundreds more — see the full list at iostindex.com.

Custom server address

If Intiface Central is running on a different machine or port, change the address in the Toy Control panel before connecting. Example: ws://192.168.1.10:12345.

Troubleshooting

  • Won’t connect — make sure Intiface Central’s server is started (green status), not just open.
  • Toy not found after scanning — confirm the toy is charged, in range (< 5 m), and paired in your OS Bluetooth settings.
  • Connection drops — keep the Intiface Central window open; minimising is fine but closing it disconnects all toys.
  • Browser blocks WebSocket — some browsers disallow plain ws:// on HTTPS pages; use the desktop app or run the site over plain HTTP.
