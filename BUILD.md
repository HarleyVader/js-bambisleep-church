# Universal Build System

## Commands

```bash
# Install all dependencies (backend + frontend)
npm run setup

# Build production app
npm run build

# Start production server
npm start

# Development with auto-reload
npm run dev
```

## Requirements

- Node.js 18+
- npm

## Universal Compatibility

These commands work on:
- Linux/Unix
- Windows
- macOS
- Docker containers
- CI/CD pipelines
