# Copilot Instructions

This repository is an Express + Socket.IO chat application with MongoDB persistence.

- Stay within the existing project structure: `src/`, `public/`, `config/`, `models/`, `routes/`, and `sockets/`.
- Use CommonJS modules unless the repository already uses ES modules consistently.
- Keep code simple and maintainable: prefer small helper functions, clear naming, and minimal dependencies.
- When adding UI or frontend code, match the current chat interface style and use existing static assets.
- Avoid adding unrelated features or files outside the chat application scope.
- If editing configuration, preserve `dotenv` usage and `.env` conventions.
- When creating or updating files, make sure they are valid and ready to run in this repository.

Use this file as the primary guidance for Copilot contributions to this repo.
